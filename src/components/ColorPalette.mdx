# Colors

## Blaze
Below is a list of colours used by the Design system for the Blaze Theme
<ColorPalette>
  <ColorItem
    title="reds"
    subtitle="Reds"
    colors={color}
  />
  {
    Object.entries(color).map(([colorName, colorValue]) => {
      return (
        <ColorItem
          key={colorName}
          title={colorName}
          subtitle={colorName}
          colors={{ [colorName]: colorValue }}
        />
      )
    })
  }
</ColorPalette>
